Higher or Lower? By Andrew Durnall

GitHub Repository

This game was created using Object Orientated Javascript and a canvas, the cars colours are randomly picked from a colour API.

The race will have a random winner each time! (Winner is announced under the race, sometimes the cars appear to be joint first but the game knows who won!).

Enjoy!
